estructura de una página web en la que con múltiples archivos JSON escritos con datos ordenados y propiedades nativas css dentro de este archivo json, con constructores que serán archivos js, se logra una interacción a tiempo real de construcción
la lógica termina siendo un index html el cual mediante un script js lee el Json, de manera secuencial y con cálculos matemáticos con fórmulas y más va creando toda la estructura separada en solo divs de una manera jerárquica permitiendo la posibilidad de poder interactuar solo con el json para cualquier cambio edición adición o mejora que se necesite, abordando desde las complejidades de efecto animación interacción hasta lo más básico que podría ser crear la estructura plantillas tarjetas secciones y más
